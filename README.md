# Where Did My Money Go - Frontend

Frontend do app **Where Did My Money Go**, um gerenciador de finanças pessoais.

---

## Descrição

- Controle de transações e categorias
- Cadastro e login de usuários
- Dashboard com saldo e histórico de gastos
- Criação, edição e exclusão de transações

---

## Funcionalidades

- Listagem de transações e detalhes
- Cadastro de novas transações
- Filtragem por categoria
- Perfil do usuário com informações básicas

---

## Como rodar

```bash
# Instalar dependências
pnpm install
# Rodar servidor de desenvolvimento
pnpm run dev
